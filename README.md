## `$ ls ~/projects/ai-agents/`

### 🤖 Autonomous GitHub Issue Solver Agent
> *An agent that reads issues, writes the fix, and opens the PR — no human in the loop.*

Built a multi-step reasoning agent that ingests a GitHub issue, forms a repair plan, writes production-ready code, verifies it, and creates a pull request autonomously. This is the kind of agentic loop that closes tickets while you sleep.

```
Stack: Python · LangGraph · OpenAI · GitHub API · Docker
```

| Capability | Detail |
|---|---|
| 🧠 Planning | Multi-step chain-of-thought before touching code |
| 🔧 Tool Use | Real GitHub API calls — clone, diff, commit, PR |
| ✅ Verification | Code review pass before PR is opened |
| 🚢 Output | Auto-generated PR with description and diff |

### 🧾 Agentic AI Job Drafter
> *Your resume, tailored to every job posting — automatically.*

An end-to-end agent that scrapes job postings with Playwright, embeds your experience into a vector store, retrieves the most relevant context via RAG, and generates a tailored resume + cover letter per application.

```
Stack: Python · LLMs · RAG · Playwright · Vector Database
```

| Capability | Detail |
|---|---|
| 🕷️ Scraping | Playwright-based job board crawler |
| 📚 RAG | Vector-retrieved profile context per posting |
| 📝 Generation | LLM-tailored resume sections + cover letter |
| ⚙️ Orchestration | Full multi-agent workflow pipeline |
